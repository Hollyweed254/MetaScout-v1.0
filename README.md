# MetaScout

MetaScout is an intelligent football analysis platform built specifically for eFootball.

Unlike traditional player databases, MetaScout focuses on explainable football analysis, helping players build stronger squads through tactical intelligence and deterministic recommendations.

## Vision

Given my players, manager, formation, playstyle, and available cards, what is the strongest squad I can build, and why?
